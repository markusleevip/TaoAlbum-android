# TaoAlbum 道相册Android客户端
## 功能说明
### 与[taostorage](https://github.com/markusleevip/taostorage)项目配置使用，实现手机相册备份到私有服务器的功能
### 在output目标已经提供Windows X64平台的可运行版本。
### 输入服务端地址，点击上传全部，将手机相册的照片全部上传到服务端，服务端使用SHA265算法对文件去重，以减少占用空间。
